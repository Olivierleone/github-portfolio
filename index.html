Department Manager

This Java-based GUI project facilitates the management of staff members and departments. The main screen offers options to add a new staff member or load a department. When adding a new staff member, users are prompted to enter details, followed by a validation prompt. Upon confirmation, a prompt displays the entered information, and upon closing, the names of added individuals are saved to a file. Loading a department allows users to select from a dropdown menu, displaying department details including name, ID, and department dean. To add a member to a department, users select the department and specify whether they're a teacher or staff member. After submission, a prompt confirms the action, and a new file stores department details along with member information. The project ensures error handling for empty inputs and incorrect member types.

Linux Dhcp + Ssh configurations

For my Linux final project, I executed a series of tasks aimed at configuring and optimizing server functionalities. Firstly, I configured DHCP services on two servers, assigning IP addresses, default gateway addresses, and DNS server IP addresses to Windows clients automatically. The servers were renamed to "Server 15" and "Server 16" respectively. Following this, I set up DHCP configurations, ensuring proper service restarts and verifications. The next phase involved establishing DNS services, installing the BIND package, mounting the DNS ISO file, and configuring DNS files meticulously to ensure accurate name resolution. Additionally, I ensured that any user logged into a Windows client could access the web server http://www.abc.com. Moreover, I implemented SSH access restrictions to only allow members of the "admins" group to log in from any host on the local network. Finally, I scripted a solution to save DHCP lease information locally on Server 15 and scheduled regular email notifications of this data using mailx and cron jobs. Through these tasks, I demonstrated proficiency in Linux server management and network administration, showcasing my skills for future endeavors.
